# 武侠传承录 (Wuxia Inheritance) 开发文档 (阶段一已完成)

**状态:** 第一版已实现并运行，内容详见下文。

---

## 技术栈

- **后端:** Node.js + Express (CommonJS)
- **前端:** Vanilla JS (ES Modules) + HTML5 Canvas
- **持久化:** JSON 文件扁平存储 (users.json / saves.json)
- **窗口:** 960×760px，水墨宣纸风格

---

## 项目文件结构

```
wuxia-inheritance/
├── start.bat                    # Windows 一键启动脚本（自动检测端口占用）
├── server/
│   ├── data/
│   │   ├── users.json           # 用户账号 (SHA-256 密码)
│   │   ├── saves.json           # 角色存档
│   │   └── static_maps.json     # 地图配置（牛家村 + 少林寺）
│   ├── server.js                # Express 服务 (含 7 个 API)
│   └── package.json
└── public/
    ├── index.html               # 前端主骨架（登录→创角→主界面三屏）
    ├── style.css                # 水墨国风样式 + 弹窗 + 战斗界面
    ├── config.js                # 地形常量、内功体系、门派、武学、公式
    ├── main.js                  # 游戏核心（状态管理、弹窗、交互、修炼）
    ├── map-renderer.js          # Canvas 瓦片渲染器（地形+NPC+传送门+主角）
    ├── combat-engine.js         # 火柴人战斗引擎（回合动画+招式+血条）
    └── index.js                 # 前端入口（登录/注册/创角/加载游戏）
```

## API 端点

| 方法 | 路径 | 功能 |
|------|------|------|
| GET | `/api/ping` | 服务器心跳 |
| POST | `/api/auth/register` | 注册 (密码 SHA-256) |
| POST | `/api/auth/login` | 登录 |
| GET | `/api/game/state` | 读取存档 |
| POST | `/api/game/save` | 保存存档 |
| GET | `/api/map` | 获取地图配置 |
| POST | `/api/game/offline` | 离线收益结算（写入存档，防重复领取） |

## 核心数值公式

### HP / 内力
```
最大气血 = 根骨 × (100 + 内功点数 × 100 + 力量 × 10) + 装备附加HP
最大内力 = 100 + 内功点数 × 50 + 装备内力
内功点数 = 内功资质点数(炼气术0/黄1/玄3/地5/天7) + 内功层数
```

### 境界（与内功点数挂钩）
| 点数 | 境界 |
|------|------|
| 1-2 | 初窥门径 |
| 3-4 | 小有所成 |
| 5-6 | 融会贯通 |
| 7-8 | 炉火纯青 |
| 9-10 | 出神入化 |
| 11-12 | 登峰造极 |
| 13-14 | 空前绝后 |
| 15-16 | 天人合一 |
| 17+ | 返璞归真 |

### 武功升级
```
所需修为 = baseCost × 2.8^(层数-1)
实际消耗 = 所需修为 × (1 - 悟性减免%) × (1 - 家传减免%)
悟性减免 = min(75%, 悟性/40)
```

## 门派系统

### 少林派
- NPC: 玄慈大师、扫地僧（少林寺地图）
- 武学: 混元功(内功)、罗汉拳(拳脚)、少林棍法(枪/棍)

### 华山派  
- NPC: 暂无（后续扩展）
- 武学: 华山心法(内功)、华山剑法(剑法)、雁行功(轻功)

### 交互逻辑
| 条件 | 按钮 | 效果 |
|------|------|------|
| 无门派 | 「拜入XX派」 | 获得该门派全部武学 Lv1 |
| 同门派 | 「请教(内功修炼)」 | 消耗30历练点，内功层数+1 |
| 不同门派 | 「判师」(红色) | 清空旧武学，拜入新门派 |

## 武学系统

### 武学分类 (8类)
内功(气)、轻功(飘)、拳脚(拳)、刀法(刀)、剑法(剑)、枪/棍(棍)、鞭法(鞭)、暗器(镖)

### 初始武学
- **炼气术**: neigongTier='lianqi'，rank=0，layers=1，角色自带

### 修炼机制
1. 在武学弹窗中点击某武学的「修炼」按钮设为当前修炼目标
2. 每 2 秒自动将修为转化为武学层数
3. 内功类武学升级时自动提升 neigongLayers
4. 挂机修炼(status) + 在线自动转化同时生效

## 装备与行囊

### 装备栏
- 武器(weapon)、护甲(armor)、饰品(trinket)
- 弹窗内可装备/卸下
- 商店购买：精铁剑(50两)、皮甲(40两)、玉佩(60两)、金疮药(15两)、回气丹(20两)

### 消耗品
- 金疮药: 恢复气血 30 点
- 回气丹: 恢复内力 20 点

## 战斗系统

### 战斗流程
- 火柴人 Canvas 动画，血条显示
- 回合制：玩家→敌人交替，每回合 ~450ms
- 6 种玩家招式名 + 3 种闪避对话
- 90% 命中率，虚弱状态攻击减半

### 战斗结果
| 结果 | 效果 |
|------|------|
| 胜利 | NPC 进入虚弱 5 分钟（HP 15%，攻击 50%） |
| 失败 | 玩家气血降至最大值的 5% |

## 地图系统

### 地形
- 山(黑框)、水(蓝)、林(绿)、平地(米白)
- 山/水不可通行
- 传送门：金色同心圆+十字星图形，无文字
- NPC/宝箱：类型图标（铺/师/敌/箱）+ 颜色区分

### 操控
- 键盘 WASD / 方向键
- 触屏虚拟方向键（pointer:coarse 自动显示）

## 打坐系统

- 人物属性弹窗内「打坐调息」按钮
- 每秒恢复 ~10 气血 + ~6.6 内力（约 10 倍速）
- 满自动停止
- 战斗失败后可快速恢复

## 启动方式

**方式一：双击 `start.bat`**
- 自动检测并关闭端口 3000
- 打开浏览器访问 localhost:3000
- 启动服务器

**方式二：命令行**
```bash
cd wuxia-inheritance/server
npm install
node server.js
# 浏览器访问 http://localhost:3000
```

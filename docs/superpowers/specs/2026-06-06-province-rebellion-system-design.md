## 1. 设计目标

在现有的朝堂博弈基础上，增加**地方治理压力**。玩家不能只关注朝中权斗，还必须处理地方叛乱。叛乱是"民心崩溃 → 统治瓦解"的核心表现。新增：

- **黄巾之乱**作为历史必然性事件（民心崩溃的终极惩罚）
- **NPC 五维属性**，让"选拔将领/特使"成为有深度的决策
- **距离系统**，让地理因素影响军事行动

## 2. 数据结构变更

### 2.1 NPC 新增五维属性

在 `NpcState` 中新增：

```csharp
public class NpcState
{
    // === 已有字段不变 ===
    public string Id { get; set; }
    public string Name { get; set; }
    // ...

    // === 新增：五维基本属性 (0-100) ===
    public int Martial      { get; set; } = 40;  // 武力：单挑/冲锋陷阵
    public int Leadership   { get; set; } = 40;  // 统帅：指挥大兵团能力
    public int Politics     { get; set; } = 40;  // 政治：治理/外交/谋略
    public int Charisma     { get; set; } = 40;  // 魅力：笼络人心/招募
    public int Ambition     { get; set; } = 40;  // 野心：反叛倾向/权力欲

    // Traits 继续存在，在五维基础上提供乘数修正
    // 例如：曹操 Martial=72, 加上 Traits["治军严整"] ×1.25 = 有效统帅
}
```

### 2.3 郡县数据结构（新增 distance 字段）

```csharp
public class Province
{
    public string Id { get; set; }            // "jizhou"
    public string Name { get; set; }          // "冀州"
    public string GovernorId { get; set; }    // 地方官 NPC ID，null = 无主
    public int Distance { get; set; }         // 距京距离（1-10单位）

    // === 叛乱状态 ===
    public bool IsRebelling { get; set; }
    public int RebellionMonths { get; set; }  // 0 = 未叛乱
    public string RebelFaction { get; set; }  // 叛军旗号

    // === 治理属性 ===
    public int LocalSupport { get; set; }
    public int Wealth { get; set; }
    public int Garrison { get; set; }
    public int DefenseLevel { get; set; }

    // === 黄巾之乱追踪 ===
    public int LowSupportStreakMonths { get; set; } // 民心<10 持续月数
}
```

## 3. 叛乱触发机制（细化版）

### 3.1 触发层级

```
每旬检查所有未叛乱郡：

┌──────────────────────────────────────────────────────────┐
│ 层级 0：黄巾之乱（必然事件）                               │
│                                                          │
│   条件：任意郡 localSupport < 10 且持续 ≥ 3 个月            │
│   （即 LowSupportStreakMonths ≥ 3）                       │
│                                                          │
│   → 必定触发！不可概率回避                                 │
│   → 触发范围：该郡 + 所有相邻郡 一次全部叛变                │
│   → 叛军旗号固定为 "黄巾军"（特殊旗帜，与普通叛乱区分）     │
│   → 全局效果额外：天下民心 -10，皇权 -15，所有郡叛乱风险 +30%│
│                                                          │
│   设计意图：这是"失败状态"的终极惩罚 —— 如果你让大郡民心     │
│   低于 10 整整 3 个月不处理，黄巾之乱就必然爆发，无法回避。   │
└──────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────┐
│ 层级 1：地方官野心叛乱（民心中等）                          │
│                                                          │
│   条件：localSupport 10-30 AND governor 存在               │
│         AND governor.Ambition ≥ 60                        │
│                                                          │
│   → 触发 = 25% 概率 / 旬                                  │
│   → 地方官被野心驱使，煽动本郡反叛                          │
│   → governor.Ambition ≥ 80 时 → 触发概率 +15%              │
│                                                          │
│   设计意图：任命高野心 NPC 到中等民心的郡 = 定时炸弹。        │
│   玩家必须在"能干的野心家"和"无能的总臣"之间权衡。           │
└──────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────┐
│ 层级 2：民心崩溃叛乱（无地方官也触发）                      │
│                                                          │
│   条件：localSupport < 10（没有满 3 个月）                  │
│                                                          │
│   → 触发 = 40% 概率 / 旬                                  │
│   → 纯民众暴动，叛军旗号为 "流民暴乱"                       │
└──────────────────────────────────────────────────────────┘
```

### 3.2 全局民心联动

```
全局 PopularSupport 下降时：
  → 所有郡的 LocalSupport 每旬额外 -1（民心传导）

全局 PopularSupport 跌破 20：
  → 所有郡 LocalSupport 每旬额外 -2（加速崩溃）
```

```

  标题：「⚡ 黄巾之乱爆发！」

         波及{相邻郡列表}。张角妖术蛊惑人心，数郡同日并起，
         天下震动！若不速派大军镇压，大汉江山危在旦夕！」

地方官叛变：
  标题：「{郡名}叛乱急报！」
  描述：「{governor.Name}举兵反叛！其以{governor.Faction}之便，
         煽动{郡名}驻军倒戈，打出"{rebelFaction}"旗号！」

民众暴动：
  标题：「{郡名}民变急报！」
  描述：「{郡名}发生大规模民变！饥民攻占县衙，开仓抢粮。
         郡中无主官弹压，乱局愈演愈烈。」
```

## 5. 平叛系统（重构）

### 5.1 平叛流程

```
1. 玩家在宣政殿选择「平叛」
2. 选择目标叛乱郡
3. 选择统兵将领（必须是朝中在洛阳的 NPC）
   → 地方官需先回京（自动，需 1 旬时间成本）
4. 决定调派兵力数
5. 执行平叛
```

### 5.2 平叛成功率公式

```
综合战力 = 将领作战力 × 0.4 + 兵力优势 × 0.3 - 距离惩罚 × 0.3

──────────────────────────────────────
将领作战力 = 基础 (Martial × 0.4 + Leadership × 0.6)
            × Trait 乘数修正
            - 贪腐惩罚

其中 Trait 修正（从 NpcTraitEvaluator 取值）：
  孔武有力 → Martial ×1.30
  有些力气 → Martial ×1.10
  治军严整 → Leadership ×1.25
  懂点兵法 → Leadership ×1.10
  不学无术 → 全属性 ×0.80

贪腐惩罚 = (Corruption / 100) × 20  点扣减

──────────────────────────────────────
兵力优势 = dispatchTroops / rebelTroops × 30
  上限 30（超过 1:1 不再增加收益）
  下限 -20

叛军兵力 = garrison × (0.5 + rebellionMonths × 0.1)

──────────────────────────────────────
距离惩罚 = Distance × 5
  司隶(0) → 惩罚 0
  冀州(3) → 惩罚 15
  荆州(5) → 惩罚 25
  解释了为什么偏远地区最难平定
```

### 5.3 判定

```
最终成功率 = 综合战力 + 随机(0, 15)

≥ 60 → 大胜：叛乱平定，全效恢复
≥ 40 → 险胜：叛乱平定，但 localSupport 仅恢复一半
< 40 → 失败：兵力损失 dispatchTroops×30%，叛乱继续

黄巾之乱特殊：大胜阈值 +15（需要 75），黄巾组织更强
```

### 5.4 平叛结果

```
大胜：
  叛乱郡恢复，localSupport = max(原值, 30)
  皇权 +5，民心 +3
  将领 Martial 永久 +2（实战经验）

险胜：
  叛乱郡恢复，localSupport = max(原值, 20)
  皇权 +3
  将领 好感 +5

失败：
  将领好感 -20
  西园军兵力 -= dispatchTroops × 0.3
  士气 -10
  rebellionMonths 继续累积
```

## 6. 安抚系统（v2 重构：四策多选 + 特使阵亡风险）

### 6.1 安抚流程

```
1. 玩家在宣政殿选择「安抚」
2. 选择目标叛乱郡
3. 选择安抚特使（必须是朝中在洛阳的 NPC）
4. 选择安抚策略组合（四策可多选，效果叠加）：

   ┌──────────┬──────────┬──────────────────────┬──────────────┐
   │  策略    │ 成功加成 │  消耗 / 前提         │  设计意图     │
   ├──────────┼──────────┼──────────────────────┼──────────────┤
   │ 离间     │  +15%    │  前提：Politics ≥ 50 │  智取         │
   │ 说服     │  +20%    │  前提：Charisma ≥ 45 │  怀柔         │
   │ 赈灾     │  +8%/500 │  国库花费 500 万/次   │  收买民心     │
   │          │  万钱    │  最多 3 次（+24%）    │              │
   │ 惩治官员 │  +15%    │  前提：皇权 ≥ 35      │  立威         │
   │          │          │  皇权 < 20 → -10%     │  但可能反效果 │
   └──────────┴──────────┴──────────────────────┴──────────────┘

   示例：离间 + 赈灾 1000 万 = +15 + 16 = +31% 加成
         说服 + 离间 + 惩治 = +20 + 15 + 15 = +50% 加成（需满足所有前提）
```

### 6.2 成功率公式

```
特使综合政治力 = Politics × 0.6 + Charisma × 0.4
                × Trait 乘数修正

Trait 修正（累计乘数）：
  经天纬地 → Politics ×1.20
  擅长民政 → Politics ×1.08
  爱民如子 → Charisma ×1.15
  老谋深算 → Politics ×1.15
  贪得无厌 → 全属性 ×0.75（叛军不信任贪官）
  刚直不阿 → Charisma ×1.10（人民信任）

最终成功率 = 特使政治力 + 策略加成之和 + 随机(0,15)
            - rebellionMonths × 5
            - 距离 × 3
            + (皇权 ≥ 50 ? +10 : 0)  ← 天子威信加持

≥ 50 → 安抚成功
< 50 → 安抚失败 → 进入 6.4 特使阵亡判定
```

### 6.3 安抚成功结果

```
叛乱郡：
  isRebelling = false, rebellionMonths = 0
  localSupport = max(原值, 40 + 特使Politics×0.2)

全局：
  皇权 +3
  使用了赈灾 → 该郡 localSupport 额外 +10
  使用了惩治 → 皇权额外 +3（杀一儆百），但所有地方官好感 -5
  使用了离间 → 特使智力经验 +2（Politics 永久 +2）
  使用了说服 → 特使魅力经验 +2（Charisma 永久 +2）
```

### 6.4 ❗ 安抚失败 → 特使阵亡风险

这是安抚系统最大的风险——失败了不只是浪费行动，**你的人可能回不来**。

```
特使阵亡概率 = 基础 40%

  ┌─────────────────────────────────────────────┐
  │ 修正因子（叠加）：                            │
  │                                             │
  │  特使 Martial ≥ 70    → -20%                │
  │  特使 Martial ≥ 50    → -10%                │
  │  特使 Martial < 30    → +15%  ← 文官很危险   │
  │  rebellionMonths      → 每月 +5%            │
  │  距离                 → 每单位 +3%           │
  │  使用了惩治官员       → +10%（激怒叛军）     │
  │  皇权 < 20            → +15%（朝廷威慑力不足）│
  └─────────────────────────────────────────────┘

  cap 在 10%-90%（再强的武将也可能被暗杀，再弱的文官也有逃跑可能）
```

**阵亡概率示例：**

| 特使 | 叛乱月数 | 距离 | 策略 | 皇权 | 阵亡率 |
|---|---|---|---|---|---|
| 曹操 (M72) 去冀州 | 6个月 | 3 | 离间+说服 | 25 | 40-20+30+9+0+0 = **59%** |
| 荀彧 (M15) 去荆州 | 4个月 | 5 | 赈灾 | 25 | 40+15+20+15+0+0 = **90%** |
| 皇甫嵩 (M75) 去兖州 | 2个月 | 2 | 说服 | 40 | 40-20+10+6+0+0 = **36%** |

### 6.5 特使阵亡后的连锁反应

```
特使确认死亡 → 触发以下所有效果：

叛乱郡：
  → 安抚彻底失败，rebellionMonths 重置为 0 但 isRebelling 保持 true
  → localSupport +10（叛军士气大振）

全局：
  → 皇权 -8（朝廷威信扫地）
  → 天下民心 -5

周边恐慌（最关键的连锁）：
  → 该郡相邻的所有郡，未来 3 个月内叛乱概率 +40%
  → 如果相邻郡已有隐性不满 → 几乎必然爆发连锁叛乱

朝局影响：
  → 特使所属派系全体 NPC 权势 -10（折损重要成员）
  → 敌对派系权势 +5（"我早说派文官去是送死"）
  → 如果特使是朝中重臣（权势≥60）→ 皇权额外 -5，民心额外 -5

继承：
  → 特使的 NPC 从游戏中永久移除（DeregisterNpc）
  → 如果特使是地方官 → 该郡变为无主状态
```

### 6.6 策略性决策总结

```
安抚 vs 平叛：

  安抚的优点：
    · 不消耗兵力（西园军不动）
    · 成功后民心恢复更高（40+ 起步）
    · 特使能力可永久成长

  安抚的风险：
    · 特使可能阵亡 → 优秀人才一去不返
    · 阵亡引发连锁叛乱 → 局面可能恶化
    · 高政治能力的特使通常武力低 → 更容易死

  最佳实践：
    · 早期叛乱（1-2 个月）：安抚首选，风险低
    · 后期叛乱（3+ 月）：阵亡率陡增，考虑平叛
    · 偏远郡县：安抚惩罚大，慎重
    · 如果特使是高武力武将（如皇甫嵩 M75）→ 阵亡风险可控
    · 如果只能派文官 → 务必搭配赈灾提高成功率，减少失败可能

  历史上，汉灵帝就曾多次派遣文官招安黄巾，结果使节被杀的案例
  比比皆是——直到皇甫嵩、卢植等武将领兵才扭转局面。这正是本系统
  想要还原的历史逻辑。
```

## 7. 距离系统总结

距离影响 **平叛成功率**、**安抚成功率**、**特使阵亡率**：

| 距离 | 郡 | 平叛惩罚 | 安抚惩罚 | 阵亡+ | 点评 |
|---|---|---|---|---|---|
| 0 | 司隶 | 0 | 0 | +0% | 京畿，天下腹心 |
| 1 | 豫州 | 5 | 3 | +3% | 天子脚下 |
| 2 | 兖州 | 10 | 6 | +6% | 一日可达 |
| 3 | 冀州 | 15 | 9 | +9% | 大军需三日 |
| 4 | 并州 | 20 | 12 | +12% | 远在边陲 |
| 5 | 荆州 | 25 | 15 | +15% | 山川阻隔 |

**战略含义：** 近处叛乱好平但通常不致命；远处叛乱难平且特使容易死在路上。派荀彧去荆州安抚 → 90% 概率回不来。玩家必须权衡"先平近处保平安"还是"先平远处防蔓延"。

## 9. 设计决策记录

| 决策 | 理由 |
|---|---|
| NPC 添加显式五维属性 | 叛乱系统需要精确的能力判定，Traits 作为乘数修正而非唯一来源 |
| 黄巾之乱为确定性事件（非概率） | 这是"失败惩罚"，不是随机事件。给玩家明确的倒计时压力 |
| 距离线性惩罚 | 简单可预测，玩家能制定清晰的战略优先级 |
| 武力+统帅共同决定战力 | 避免"高武力低统帅"的猛将和"高统帅低武力"的儒将都被低估 |
| 地方官需先回京才能领兵 | 符合历史逻辑，也让玩家在任命地方官时多一层考量 |
| 安抚四策多选可叠加 | 让玩家有策略组合空间，不是简单的单按钮选择 |
| 赈灾有上限（1500万/+24%） | 防止无限砸钱通关，保留合理上限 |
| 惩治官员受皇权制约 | 皇权低时"惩治"是玩火，符合"政令不出宫门"的设定 |
| 特使阵亡触发连锁叛乱 | 让失败成本远大于浪费一个行动——可能毁掉整个局面 |
| 高武力特使阵亡率低 | 鼓励玩家派遣武将安抚，而非只用文官——符合历史 |

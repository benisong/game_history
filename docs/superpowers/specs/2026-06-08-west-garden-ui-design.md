> **状态：📋 设计阶段 — 未实现**

## 1. 设计结论

西园不应只是“起驾后在正文里看一段描述 + 旧右侧按钮”，而应成为玩家经营天子私库、整顿亲军、召见西园武臣的专属基地界面。

一句话目标：**主界面像御案，朝会像朝堂，黄门密札像情报司，西园像天子私军与私库的军务密署。**

它承担三件事：

1. **西园军势总览**：西园新军人数、上限、士气、忠诚、基础军饷、当前所在地。
2. **私库与军费决策**：鬻官入私库、阅兵发饷、募兵补军都在这里集中执行，清晰展示代价。
3. **西园人物关系**：蹇硕、曹操、张让等与西园相关 NPC 在这里出现，玩家可以召见、查看详情、选择经办人。

## 2. 当前实现审计

当前代码已经具备西园规则和部分 UI 能力：

- `GameState.WestGardenArmy`：
  - `Size = 8000`
  - `BasePayPerTurn = 120`
  - `Morale = 55`
  - `Loyalty = 50`
- `GameEngine.TravelToLocation("西园")`：允许起驾到西园，并写入 Chronicle。
- `ExecuteQuickAction("sell_office")`：仅在西园执行，私库 +1000，皇权 -3。
- `ExecuteDrillArmyActionWithOfficer(paidAmount, officerId)`：仅在西园执行，消耗私库，按经办人贪腐/特质影响实际入营金额、士气、忠诚、皇权。
- `ExecuteRaiseWestGardenTroopsAction(troops)`：仅在西园执行，1000 人步进，上限 12000；每 1000 人国库 -300、民心 -1、士气 -1。
- 前端已有：
  - `DoTravel("西园")` 转场正文。
  - `DoQuickAction("sell_office")`。
  - `DoArmyDrillAction(amount)`，默认根据简单条件选择经办人。
  - `ShowRecruitArmyDialog()`，已有募兵数量 SpinBox 和代价预览。

但玩家体验仍有明显问题。

### P0 / P1

暂无明显会导致错误行为的问题。

### 影响游戏性的 P2

1. **西园场景入口后没有专属面板**
   - 起驾到西园后，玩家主要看到正文；可执行动作仍依赖旧 `RightPanel/Ministers` 节点。
   - 但主界面已极简化并隐藏 RightPanel，导致西园动作缺乏新的、稳定的可见入口。

2. **西园信息分散**
   - 西园军人数/士气/忠诚在情报面板顶部出现，但西园本场景没有集中展示。
   - 私库、国库、军费、民心代价散落在不同弹窗或叙事文本里。

3. **阅兵发饷交互过旧**
   - 仍依赖旧 `ArmyDrillPopupPanel` + LineEdit 输入金额。
   - 经办人由前端硬编码自动选择：曹操好感 > 50 则曹操，否则张让权势 > 75 则张让，否则蹇硕。
   - 玩家看不到经办人的贪腐、特质、实际入营金额风险，缺少策略感。

4. **募兵补军弹窗较好，但不属于整体西园界面**
   - `ShowRecruitArmyDialog()` 已有 SpinBox 与代价预览。
   - 但应该作为西园右栏“军务处置”的一项，而不是孤立按钮。

5. **西园 NPC 没有场景化**
   - 蹇硕是西园上军校尉，曹操可任军事，张让掌私库/卖官，但当前没有“西园班列/校尉帐前”的展示。
   - 玩家难以理解为何选择某人经办发饷会有不同后果。

6. **主界面信息边界不一致**
   - 已确认原则：主界面只放御案器物入口，场景信息只在具体场景显示。
   - 西园应像朝会/情报一样有专属大面板，而不是恢复旧右侧数据栏。

## 3. 第一轮实现范围

第一轮目标：**不新增后端规则，只重构西园 UI 信息架构与现有动作入口。**

### 做

- 新增 1100×640 左右的“西园精舍”专属大面板。
- 点击起驾到西园后，在正文保留转场故事，同时玩家可通过新入口打开西园面板。
- 推荐：当玩家已在西园时，顶部御案按钮区域额外显示或复用一个“🏯 西园”入口；也可以在起驾完成后自动弹出西园面板。
- 面板展示：
  - 西园军人数 / 上限 / 士气 / 忠诚 / 基础军饷。
  - 私库、国库、民心、皇权等与西园动作相关的少量数值。
  - 西园相关人物：蹇硕、曹操、张让。
- 操作分组：
  - **私库经营**：鬻官纳钱（调用 `ExecuteQuickAction("sell_office")`）。
  - **阅兵犒赏**：选择经办人 + 输入/选择发饷金额（调用 `ExecuteDrillArmyActionWithOfficer(amount, officerId)`）。
  - **募兵补军**：复用现有募兵规则（调用 `ExecuteRaiseWestGardenTroopsAction(troops)`）。
  - **召见校尉**：打开 NPC 详情，第一轮只调用 `ShowMinisterDetails(...)`。
- 执行后刷新面板与主界面，并把后端 `StoryText` 写入主故事区。
- 保持西园动作只能在 `CurrentLocation == "西园"` 执行；若不在西园点击入口，则弹出起驾确认。

### 不做

- 不新增军队训练新公式。
- 不新增西园官职任免系统。
- 不新增私库账本或长期军饷自动支出。
- 不新增真实 AI 校尉对话。
- 不做复杂 2D 校场/人物立绘站位。
- 不把黄门密札的天下州郡信息搬进西园。

## 4. 推荐布局

```text
┌──────────────────────────────────────────────────────────────┐
│ 西园精舍 · 天子亲军密署                 光和7年 · 184年4月上旬 │
│ 所在地：西园   私库1200万   国库8000万   皇权25   民心28       │
├───────────────┬──────────────────────────────┬───────────────┤
│ 西园军势       │ 校场与私库                    │ 军务处置       │
│               │                              │               │
│ 新军 8000/12000│ 【军势研判】                  │ 【私库经营】   │
│ 士气 55        │ 兵未满编，可募兵补军。         │ [鬻官纳钱]     │
│ 忠诚 50        │ 士气中等，发饷可稳军心。       │               │
│ 基础军饷 120万 │ 忠诚未稳，低饷可能激怒军士。   │ 【阅兵犒赏】   │
│               │                              │ 经办：蹇硕 ▼   │
│ 校尉帐前       │ 【近日西园记】                 │ 发饷：1000万   │
│ 蹇硕 上军校尉   │ 陛下轻车简从入西园……           │ [发内帑犒军]   │
│ 曹操 典军可任   │                              │               │
│ 张让 中官掌财   │                              │ 【募兵补军】   │
│               │                              │ 募兵：2000人   │
│               │                              │ [下诏募兵]     │
│               │                              │ [离开西园]     │
└───────────────┴──────────────────────────────┴───────────────┘
```

### 4.1 顶部标题区

显示：

- `西园精舍 · 天子亲军密署`
- 当前时间：`光和7年 · 184年4月上旬`
- 关键西园相关数值：所在地、私库、国库、皇权、民心。

这些数值不是“完整天下数据”，而是西园动作直接影响的资源与风险。

### 4.2 左栏：西园军势

显示两块内容。

**军势卡：**

- 新军人数：`8000 / 12000`
- 士气：`55 / 100`
- 忠诚：`50 / 100`
- 基础军饷：`120 万/旬`
- 风险标签：
  - 士气 < 35：`军心浮动`
  - 忠诚 < 35：`忠诚堪忧`
  - 人数 < 6000：`兵力不足`
  - 人数 >= 12000：`已近满编`

**校尉帐前：**

固定列出当前初始可用的西园相关 NPC：

- 蹇硕：西园上军校尉，默认亲军代表。
- 曹操：可任军事，经办廉洁/能力更强但政治上未必完全受控。
- 张让：中官掌财，可能经手私库但贪腐风险高。

每个 NPC 行可点击打开详情。

### 4.3 中栏：校场与私库

显示解释型信息，不放过多按钮。

内容包括：

- **军势研判**：根据兵力/士气/忠诚生成 3–5 条建议。
- **私库账目提示**：说明发饷消耗私库、募兵消耗国库、鬻官增加私库但损皇权。
- **近日西园记**：展示最近 Chronicle 中包含“西园/募兵/阅兵/鬻官/平叛”的数条记录；没有记录时显示场景氛围文本。

### 4.4 右栏：军务处置

右栏只放会改变游戏状态的按钮和输入控件。

#### 私库经营

- 按钮：`鬻官纳钱`
- 调用：`DoQuickAction("sell_office")`
- 预览：`私库 +1000 万，皇权 -3`
- 风险提示：买官鬻爵会损害朝廷声望，后续可引发地方治理隐患。

#### 阅兵犒赏

- 经办人选择：蹇硕 / 曹操 / 张让。
- 发饷金额：SpinBox，建议步进 100 万，默认 1000 万，上限为当前私库。
- 预览：
  - 私库支出。
  - 经办人贪腐风险（按 `Corruption` 显示低/中/高）。
  - 可能影响：士气、忠诚、皇权。
- 调用：`ExecuteDrillArmyActionWithOfficer(amount, officerId)`。

第一轮不必在前端完整复刻后端公式，只需给“贪腐风险/大致作用”的提示；实际结果以后端返回 `StoryText` 为准。

#### 募兵补军

- 募兵人数：SpinBox，1000 人步进，默认 2000。
- 上限：`12000 - 当前西园军 Size`。
- 预览沿用现有规则：
  - 每 1000 人：国库 -300 万、民心 -1、士气 -1。
- 调用：`ExecuteRaiseWestGardenTroopsAction(troops)`。

#### 召见校尉

- `召见蹇硕`
- `召见曹操`
- `召见张让`

第一轮只打开 `ShowMinisterDetails(id)`；后续再接单 NPC 对话。

## 5. 入口规则

### 5.1 从起驾进入

玩家点击主界面 `🐎 起驾` → 选择 `西园`：

1. 调用 `DoTravel("西园")`。
2. 主故事区显示既有西园转场文案。
3. 第一轮推荐自动弹出西园精舍面板，减少“到达后不知道能做什么”的问题。

### 5.2 从御案入口进入

可选方案：当 `CurrentLocation == "西园"` 时，在主界面御案按钮区域显示：

```text
🏯 西园
```

点击打开西园精舍面板。

如果不在西园点击该入口，则弹出确认：

```text
【黄门急奏】
西园乃天子亲军密署，陛下尚未起驾。

[起驾西园] [取消]
```

第一轮可先不新增常驻按钮，而是在起驾到西园后自动打开面板；但最终建议二者都有。

## 7. 与现有界面的关系

### 主界面

保持极简：不恢复旧 RightPanel，不常驻显示 NPC/硬数值。

### 朝会

朝会只负责议题讨论与圣裁。涉及西园整军时，可跳转到西园：

- 当前朝会已有 `travel_garden` 裁断动作：调用 `DoTravel("西园")`。
- 后续可在 `DoTravel("西园")` 后自动打开西园面板，使朝会裁断形成闭环。

### 黄门密札

黄门密札展示西园军势摘要，但不负责西园军内部经营。

- 密札：看天下、看州郡、处理地方。
- 西园：养亲军、用私库、召校尉。

## 8. 第一轮验收标准

完成第一轮实现后，应满足：

1. 在西园可以打开 1100×640 左右的西园精舍面板。
2. 面板左栏显示西园军人数、士气、忠诚、基础军饷、相关 NPC。
3. 面板中栏显示军势研判与西园相关记录。
4. 面板右栏可以执行：
   - 鬻官纳钱。
   - 阅兵发饷（选择经办人 + 金额）。
   - 募兵补军（选择人数）。
   - 召见蹇硕/曹操/张让查看详情。
5. 执行动作后：
   - 主故事区显示后端返回叙事。
   - 面板刷新最新数值。
   - `UpdateUI()` 正常运行。
6. 不在西园时尝试打开西园面板，会提示起驾西园，而不是直接执行军务。
7. `dotnet build DonghanFrontend.csproj -v minimal` 通过。
8. `dotnet test DonghanEngine.Tests.csproj` 通过。

## 9. 后续增强路线

### 第二轮：更有策略性的军务

- 阅兵发饷前端展示更准确的公式预览。
- 发饷金额推荐档位：薄赏 / 常赏 / 厚赏 / 重赏。
- 经办人风险卡：贪腐、特质、能力、派系影响。

### 第三轮：西园校尉与人事

- 召见校尉接入 `IMinisterAgent` 单 NPC 对话。
- 新增西园职务任免：上军校尉、中军校尉、典军校尉等。
- 校尉掌军过久可能形成尾大不掉风险。

### 第四轮：亲军自动军饷与军纪

- 每旬自动消耗私库或国库维持西园军。
- 欠饷导致士气/忠诚下降。
- 军纪事件：扰民、哗变、请赏、暗通外戚。

### 第五轮：与地方平叛联动

- 西园军频繁出征导致留守不足，洛阳安全下降。
- 平叛归来的将领获得声望与军中拥戴。
- 亲军规模过大可能刺激外戚/士族反弹。
